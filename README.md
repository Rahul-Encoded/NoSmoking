# No Smoking - Quit Smoking Tracker

A comprehensive dashboard designed to help users quit smoking by tracking their progress, health improvements, and financial savings.

## Features

- **Dashboard Overview**: Visualizes key metrics like days smoke-free, money saved, and life regained.
- **Calendar Tracking**: Interactive calendar to log daily smoking status (Smoked / Did Not Smoke).
- **Financial Savings**: Calculates money saved based on cigarettes not smoked.
- **Health Improvements**: Tracks health milestones and life expectancy regained.
- **User Authentication**: Secure sign-in and data persistence using Clerk and Prisma.
- **Responsive Design**: Built with a modern, responsive UI using Tailwind CSS.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Database**: [PostgreSQL](https://www.postgresql.org/) (via [Prisma](https://www.prisma.io/))
- **Authentication**: [Clerk](https://clerk.com/)
- **UI Components**: Custom components + [Shadcn UI](https://ui.shadcn.com/) (assumed based on structure)

## Future Plans

- **Investment Calculator**: The money you will save from not smoking can be invested. We plan to add features to project potential returns if these savings were invested in stocks, mutual funds, or other instruments.
- **Community Features**: Connect with other users for support and motivation.
- **Gamification**: Badges and achievements for reaching milestones.
